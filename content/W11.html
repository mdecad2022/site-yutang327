建立 TCExam 題庫
各分組如何建立 TCExam 線上考試系統的題庫?
  1. 啟動可攜 Python 程式系統後, 在命令列中輸入 leo, 開啟 Leo Editor.
  2. 下載 tcexam_network_setting_cadlab.leo, 以 Leo Editor 開啟.
  3. 了解如何使用 Leo Editor.
  4. 各組組員根據下列規劃出題:
題庫建立時間 - header 標註:

XML

<?xml version="1.0" encoding="UTF-8" ?>
<tcexamquestions version="14.8.5
">
<header lang="en" date="2022-11-10 14:02:11"></header>
@others
</tcexamquestions>
@language xml
@tabwidth -4
根據命題時間, 修改上列 header 內容.
接下來修改題庫模組名稱 - module -name:

XML

<module>
    <name>network_setting</name>
    <enabled>true</enabled>
    @others
</module>
module 內的 name 請改為 cp2022_ag1_w12, 表示該題目模組名稱為 cp2022 課程 a 班第一組在 w12 週所建立的題庫.
接下來, 修改有關題庫主題 - subject - name:

XML

<subject>
<name>cadlab</name>
<description>有關電腦輔助設計室網路相關設定</description>
<enabled>true</enabled>
@others
</subject>
subject 標註下的 name 表示該題庫的主題, 請將 subject 下的 name 標註內容改為 w12. 並在其下的 description 標註說明此題庫的相關測驗內容.
接下來則開始建立試題:

XML

<question>
<enabled>true</enabled>
<type>multiple</type>
<difficulty>1</difficulty>
<position></position>
<timer>0</timer>
<fullscreen>false</fullscreen>
<inline_answers>false</inline_answers>
<auto_next>false</auto_next>
<description>請根據機械設計工程系綜一館八樓的電腦輔助設計室電腦與網路相關設定, 選出下列正確的敘述.</description>
<explanation>在電腦輔助設計室上課時, 將網路線直接連接到系主幹, 在正常情況下, 可以得到最大的連外頻寬.</explanation>
@others
</question>
question 標註中, 出題者僅需要將複選題的說明文字放入 description 標註中.
最後, 出題者需要提供 8-12 個選項, 其中必須 true 與 false 選項各半.
題目選項 - answer - isright - description:

XML

<answer>
    <enabled>true</enabled>
    <isright>true</isright>
    <position></position>
    <keyboard_key></keyboard_key>
    <description>上課時段之所以只選用 IPv6 網路協定, 是為了取得對外網路連線最大頻寬.</description>
    <explanation></explanation>
</answer>
若 isright 標註中為 true 表示該 description 選項敘述正確, 若 isright 標註中為 false, 則表示該 description 選項敘述錯誤.
